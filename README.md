🔍 AI LinkedIn Profile Analyzer

> An AI-powered SaaS web application that analyzes your LinkedIn profile like a senior recruiter — giving you a score, strengths, weaknesses, ATS keywords, and a full improvement roadmap.









---

✨ Features

🔐 User Authentication — Register & Login with secure session management

🤖 AI-Powered Analysis — Claude AI analyzes your profile like a real recruiter

📊 Profile Score (0–100) — Instant ATS-optimized score with visual ring

💪 Strengths & Weaknesses — Detailed section-by-section breakdown

✍️ AI Rewritten Headline & About — Optimized copy you can directly use

🎯 Missing Skills Detection — Industry keywords you're missing

💼 Matching Job Roles — Roles you qualify for based on your profile

🗺️ Improvement Roadmap — Step-by-step action plan

📋 Reports History — All your past analyses saved and accessible

🌙 Dark Glassmorphism UI — Modern SaaS design (Notion/ChatGPT style)

🚀 Getting Started

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/linkedin-ai-analyzer.git  
cd linkedin-ai-analyzer

2. Install dependencies

npm install

3. Run locally

npm run dev

Open http://localhost:5173 in your browser.


---

📦 Build for Production

npm run build


---

☁️ Deploy to Vercel (Free)

1. Push this repo to GitHub


2. Go to vercel.com → Import Project


3. Select your GitHub repo


4. Click Deploy ✅



Your app will be live at https://your-app-name.vercel.app

💡 How to Use

1. Register with your name, email & password


2. Go to AI Analyzer page


3. Open your LinkedIn profile → Copy your headline, About, experience & skills


4. Paste the text into the analyzer


5. Click Analyze Profile


6. Get your full AI report in seconds 🚀
